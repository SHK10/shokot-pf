RPA Web Application

Using puppeteer, Node.js, Docker

[ Site Link ]

Service Overview

I've created an RPA web application that automates the process of registering on a daily basis. The RPA tool was created as a WebApplication because it can be run from any environment without depending on the device, and it can also be used by other members. To run the browser automation on a PaaS environment like Heroku, I use the Puppeteer library with built-in driver and Chromium. Furthermore, I've extended the application to be containerized so that it can be deployed independently of the environment. This application is triggered by an API request created in Koa.js (Node.js), which launches a headless browser and executes the daily registration process on the back-end. The execution result is saved as a capture and displayed on the front side.

Skills

Front-end : HTML, CSS, Javascript
Back-end / DB : Node.js (Koa.js), Typescript, Puppeteer
Environments : Heroku
Others : Git, Docker
