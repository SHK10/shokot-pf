Web-Robot Remote Call System

Remote support using video call and control robot

Project Overview

I was responsible for the design, development, and maintenance of a WebRTC video calling application for help desk users (operator). This system is remake as a web application from iOS application developed in France. I was involved in this project from the first release and was mainly responsible for the implementation of the call screen where the operator talks to robot. At first release, this system has a problem that it takes too much time about 15 seconds to start calling. We have re-examined the way in which we set up the call connection and have succeeded in reducing the time taken to initiate the call to around 8 seconds. 【Reference Link】
